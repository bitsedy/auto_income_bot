# 🤖 Income Bot Pro v2.0

**Fully autonomous, self-improving passive income engine. Zero funding required. Zero maintenance after setup.**

## What This Does

Every day at 9am UTC, this bot:
1. **Scrapes** Google Trends, Reddit, HackerNews & Product Hunt for trending topics
2. **Scores** each topic on commercial intent, search volume, and affiliate potential
3. **Writes** 1,200-1,800 word SEO articles using a 4-stage AI pipeline (Outline → Write → Critic → SEO)
4. **Embeds** Amazon affiliate links naturally with tracked short URLs
5. **Publishes** to Dev.to, Hashnode, Medium, Twitter/X, LinkedIn, and Substack
6. **Generates** Gumroad digital products every 7th run
7. **Self-optimizes** weekly — learns which niches earn money and adjusts weights
